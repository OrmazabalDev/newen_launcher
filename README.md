# Newen Launcher

Launcher moderno y rapido para Minecraft. Gestiona instancias, mods y rendimiento en un solo lugar.

## Descargas
Ve a la seccion **Releases** y descarga:
- `Newen Launcher_1.1.x_x64-setup.exe` (recomendado)
- `Newen Launcher_1.1.x_x64_en-US.msi`

## Cambios en 1.1 (Early Access)
Esta version se enfoca en calidad interna y limpieza. No agrega nuevas funciones, pero mejora la base del launcher.

**Que cambio y por que**
- **Codigo mas ordenado:** se dividieron pantallas grandes en componentes y hooks mas chicos para facilitar mantenimiento.
- **Estructura mas clara:** backend y UI quedaron separados por dominios, lo que reduce errores y hace mas facil agregar cambios futuros.
- **Estilo consistente:** se estandarizo formato de codigo, linting y tipado estricto para detectar fallos antes.
- **Instalador mas profesional:** se mejoro el look del instalador y se preparo la base para una firma digital futura.

## Caracteristicas
- Instancias con Vanilla, Forge, NeoForge y Fabric.
- Catalogo integrado para mods, modpacks, resource packs, datapacks y shaders (Modrinth).
- Instalacion directa desde Modrinth.
- Importacion de datapacks por ZIP.
- Optimizaciones predefinidas para mejorar rendimiento.
- Gestion de skins y capas offline.
- Diagnostico con reportes de logs para soporte.
- Presencia en Discord.

## Requisitos
- Windows 10/11 x64.
- Conexion a internet para descargas y actualizaciones.
