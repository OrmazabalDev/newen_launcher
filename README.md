# Newen Launcher

Newen Launcher es un launcher moderno para Minecraft enfocado en rendimiento, orden y una experiencia limpia para jugar.  
Early Access: estamos mejorando rapido y escuchando feedback en el Discord oficial.

## Descargas
Las builds oficiales estan en **Releases**:
- `Newen Launcher_1.1.x_x64-setup.exe` (recomendado)
- `Newen Launcher_1.1.x_x64_en-US.msi`

## Por que Newen
- Menos pasos para jugar, mas claridad en instancias.
- Catalogo integrado y gestion simple de contenido.
- Optimizaciones listas para usar.

## Funciones clave
- Instancias con Vanilla, Forge, NeoForge y Fabric.
- Catalogo Modrinth (mods, modpacks, resource packs, shaders, datapacks).
- Instalacion directa desde el catalogo.
- Skins offline y diagnosticos con reportes.

## Estado Early Access
Este proyecto esta en etapa temprana.
- El instalador aun no tiene firma digital (Windows puede mostrar advertencia).
- No hay telemetria ni trackers publicitarios.
- El soporte oficial y la comunidad viven en Discord.

## Soporte y comunidad
Si tienes dudas o problemas, el canal oficial es **Discord**.  
Ahí publicamos changelog, anuncios y estado del launcher.
